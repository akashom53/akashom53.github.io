Classroom Emergency

Quick help using direct push notifications from app.

Classroom emergency app for BlackBeltHelp clients. Push notification based quick help app. User can quickly ask for help regarding classroom issues like electricity failure, projector malfunction, etc, and the issue is broadcasted from app itself to all other users registered as professionals. The professional user’s app starts ringing like a phone call and he/she can then accept/reject it based on which further logic plays.

Features

  • Urban Airship push notifications
  • Silent notification handling based in user location and role
  • Parallel reporting to cloud

Svarochi

BLE based light control application.

IOT app for controlling BLE enabled lights. Built for an upcoming Light manufacturer. App searches for BLE devices with UUID filter so as to discover only the Svarochi Lights. The lights work on mesh technology, so one app is connected to any one nearby light, we send a packet with uuid inside it and the packet in bounced about within the mesh until TTL expires. This way, the lights of a whole house can be controlled. I created the android app end to end. The iOS app was started by another team member during POC phase, but was given to me when actual product development started.

Features

  • Social Login
  • BLE discovery and data exchange
  • Complex business logic to handle the frequent connection drops in BLE
  • Scene creation was implemented in app (was not supported in the Light firmware)
  • OTA implementation
  • Database sharing between Android and iOS app
